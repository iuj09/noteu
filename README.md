## 프로젝트 소개

- Note + Education
- 학습 관리와 교육이 이루어지는 교육 플랫폼

## 개발 환경
- Front : Thymeleaf / BootStrap
- Back : JAVA17 / Spring Boot 3.1.5 / Spring Security / Spring Data JPA
- DB : AWS RDS (MySQL)
- 협업 툴 : Notion
- 배포 환경 : AWS EC2 (Ubuntu 20.04 LTS)
